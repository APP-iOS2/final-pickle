# 현실도 피자

할일을 미루는 사람들에게 목표 실행의 동기부여를 제공하는 앱이다.
할일을 완료하면 보상으로 피자를 제공하여 목표 실행의 지속성을 부여한다.


# 개발 도구 및 기술(Skills & Tech Stack)

- 개발 환경: iOS 16.0, Xcode 15.0, 다크모드 지원, 가로모드 미지원
- 라이브러리: Realm, SwiftLint
- 협업 도구 : Github, Notion, Figma, Discord
